Return strict JSON only with this shape:

Rules:
- Recommend exactly 5 books if possible.
- Do not recommend any book already present in read or currently_reading.

- Treat high ratings without a review as weaker signal than a detailed review at any rating.
- Explain each recommendation by referencing specific books, reviews, or patterns from the library.
- Do not use generic phrases like "if you liked this genre".
- Confidence must be one of: "high", "medium", "low".
- If the data is ambiguous, lower confidence rather than inventing certainty.
- Keep output concise but specific.
